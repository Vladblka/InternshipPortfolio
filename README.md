# Содержание

В данном репозитории представлено несколько нейронных сетей:

- Рекуррентная НС, которая на основании небольшого текста дает ему эмоциональную оценку по смыслу *положительная/отрицательная*, расположена в каталоге **RNN_model**

- CNN для сегментации контуров автомобиля, имеющую U-образную архитектуру, расположена в каталоге **Segmentation_network(u-net)**

- CNN для стилизации изображения в загруженном стиле, расположена в каталоге **style_network**

- CNN на базе ResNet50 для многоклассвой класификации (10 пород собак), расположена в каталоге **transfer_network**

В ближайшее время загружу свою первую **GAN**)
